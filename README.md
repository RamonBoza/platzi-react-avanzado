# platzi-react-avanzado

steps:

    git clone https://github.com/midudev/curso-platzi-react-avanzado
    npm init -y

then we open the project on our beloved IDE

from there we install some dependencies

    npm i webpack wepack-cli --save-dev

now building system

    npm run dev
